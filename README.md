1. Descripción General
-Este proyecto desarrolla un dashboard interactivo en Power BI Desktop cuyo objetivo es:
“Proveer una vista ejecutiva del desempeño comercial de Superstore, analizando ventas, rentabilidad y comportamiento de clientes por región, categoría y período de tiempo, con el fin de facilitar la toma de decisiones estratégicas.”

2. Arquitectura del Modelo de Datos
Modelo estrella con:
- Tabla de hechos: Orders, 2023, 2024
- Tablas dimensión: Clientes, Productos, Locación, Tiempo

3. Proceso ETL – Power Query
Transformaciones aplicadas:
- Limpieza y tipificación
- Eliminación de columnas
- Filtrado de registros
- Columnas condicionales
- Combinación y anexado de consultas
- Expansión de dimensiones
- Construcción de tabla calendario

4. Medidas DAX
- Venta Total
- Margen Bruto
- Número de Operaciones
- Número de Clientes
- Ticket Promedio

5. Diseño del Dashboard
Incluye:
- KPI Cards
- Línea de Ventas por Mes
- Barras por Sub-Categoría
- Rentabilidad por Región
- Top Ciudades y Peores Ciudades

6. Flujo de Trabajo
- Extracción desde Excel
- Transformación en Power Query
- Modelado estrella
- Cálculo de KPIs en DAX
- Visualización ejecutiva
